# GearSolid
Uma ferramenta didática para visualizar o funcionamento de engrenagens sem necessidade de componentes físicos.

O projeto foi feito com o objetivo de auxiliar no ensino de conceitos básicos de mecânica, permitindo que estudantes e entusiastas possam
entender conceitos básicos de MCU e transmissão de movimento através de uma interface interativa.

## Pré-requisitos:

- Ter o Java 25 ou superior instalado.
- Observação: não é necessário instalar o JDK completo. Basta ter o Java Runtime (JRE) compatível com a versão do projeto.


## Como executar:

- Abra: `SimulacaoEngrenagens/run/RodarSimulacaoEngrenagens.bat`

## Funcionalidades atuais do projeto:

- Permite ao usuário definir parâmetros das engrenagens, como número de dentes, tamanho e velocidade.
- Simula o funcionamento de engrenagens simples, exibindo na interface como diferentes engrenagens interagem.
- Atualiza em tempo real o movimento das engrenagens, incluindo alterações de velocidade e direção de rotação.
- Possui uma interface gráfica intuitiva e de fácil uso.

## Funcionalidades futuras planejadas:

- Implementação de sistemas de engrenagens mais complexos, incluindo engrenagens planetárias.
- Inclusão de opções para salvar e carregar configurações de simulações.
- Melhoria da interface gráfica com mais opções de personalização e visualização.
- Simulação de trabalho sob carga e análise de eficiência das engrenagens.
